# FotoLinkedin IA

Projeto de estudo e pratica com Next.js, API e automacoes com n8n.
Este repositorio ainda esta no comeco e serve para registrar o progresso do desenvolvimento.

## Objetivos

- Evoluir o front-end com Next.js
- Integrar e testar APIs
- Criar automacoes usando n8n
- Documentar o progresso ao longo do tempo

## Status

Em desenvolvimento inicial, com primeiro layout funcional.

## Progresso

- Header com logo, navegacao e altura h-16
- Hero com CTA e exemplos usando fotos reais (`foto1.jpg`, `foto2.jpg`, `foto3.jpg`)
- Upload de foto com area clicavel/drag, preview, botao de remover e CTA "Gerar foto profissional"

## Como rodar

```bash
npm run dev
```

Depois, acesse `http://localhost:3000`.
